Curso de Android con kotlin

¡Bienvenido a bordo!

Aprende todo lo necesario para crear tu propia aplicación para el sistema operativo Android. Además de ello, aprende las mejores prácticas para garantizar la eficiencia y fácil mantenimiento de tu proyecto.

El sistema operativo Android está presente en cada rincón del planeta. De hecho, según la consultora IDC, a fines de 2017 la penetración de este sistema operativo alcanzó al 85%. Además, distintos estudios demuestran que los usuarios pasan cada vez más tiempo utilizando aplicaciones móviles.

Conoce a Andy ...

Este curso fue iniciado en 2019 utilizando la versión 3.2 de Android Studio

¡Nota importante!

A lo largo del curso se utilizará Kotlin como lenguaje de desarrollo en lugar de Java.

Las razones se darán a conocer en la sección de Introducción.
